# Calamba Medical Center

Interactive landing page mockup for **Calamba Medical Center (CMC)** — Laguna, Philippines.

Built for ClickShare Media / Dowan client presentation.

## Features

- Sticky nav with mobile drawer
- Hero, trust bar, centers of excellence
- Filterable services, animated stats
- HMO marquee, about, contact form
- Floating emergency call button

## Tech Stack

- Next.js 16 · React 19 · TypeScript
- Tailwind CSS v4 · Shadcn/ui
- Framer Motion · Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy

Deploy as a standalone Vercel project.
